# tiktok-api

FastAPI + yt-dlp endpoint for TikTok profile analysis.

## Endpoint
- `GET /api/analyze?username=<username>`

## Deploy to Vercel
1. Push this folder content to a new GitHub repo named `tiktok-api`.
2. Import the repo to Vercel.
3. Deploy.

Expected URL example:
- `https://tiktok-api.vercel.app/api/analyze?username=kaycee`
